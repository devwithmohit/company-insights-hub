# Indian Tech Companies - Salary & Work Culture Analysis

## Overview

This repository provides a detailed analysis of the first 10 tech companies in India, focusing on:

- **Company Type** (Service/Product-based)
- **Locations & Work Modes** (Remote, Hybrid, In-office)
- **Salary Ranges** (Freshers & Experienced)
- **Common Job Roles**
- **Tech Stack**
- **Company Size & Hiring Frequency**
- **Employee Reviews & Work Culture**
- **Important Notes & Research Recommendations**

The data is compiled from public sources (Glassdoor, Indeed, PayScale, employee reviews) as of November 2025. It aims to help job seekers and professionals make informed decisions about career opportunities in the Indian tech industry.

## File Structure

- `company_details_table.md`: Contains detailed information and analysis for each company.

## How to Use

- Browse the `company_details_table.md` file for insights on salary, work culture, and tech stack.
- Use the notes and disclaimers section for tips on verifying company information and preparing for interviews.

## Disclaimer

All information is based on publicly available data and estimates. Please verify details directly with companies before making career decisions.
